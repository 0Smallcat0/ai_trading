# 當前檔案結構說明

## 項目根目錄結構

```
auto_trading_project/
├── src/                    # 主要源代碼目錄
│   ├── ui/                 # 用戶界面模組
│   │   ├── web_ui.py      # Streamlit主應用入口
│   │   └── pages/         # 各功能頁面
│   ├── core/              # 核心業務邏輯
│   ├── data_sources/      # 數據源模組
│   ├── strategies/        # 交易策略
│   ├── risk_management/   # 風險管理
│   └── ...
├── data/                  # 數據存儲目錄
│   ├── enhanced_stock_database.db  # 主數據庫
│   └── ...
├── config/                # 配置文件
├── docs/                  # 文檔目錄
├── tests/                 # 測試文件
├── scripts/               # 工具腳本
├── requirements-*.txt     # 依賴文件
└── pyproject.toml        # 項目配置
```

## 主要模組說明

### 1. UI模組 (src/ui/)
- **web_ui.py**: Streamlit主應用，包含12個功能分類
- **pages/**: 各功能頁面實現
  - data_download.py: 數據下載功能
  - portfolio_management.py: 投資組合管理
  - risk_management.py: 風險管理
  - 等等...

### 2. 核心模組 (src/core/)
- **real_data_integration.py**: 真實數據整合服務
- **portfolio_manager.py**: 投資組合管理器
- **risk_controller.py**: 風險控制器

### 3. 數據源模組 (src/data_sources/)
- **real_data_crawler.py**: 股票數據爬蟲
- **taiwan_stock_list_manager.py**: 台股清單管理
- **yahoo_finance_crawler.py**: Yahoo Finance數據源

### 4. 數據庫 (data/)
- **enhanced_stock_database.db**: SQLite主數據庫
- 包含股票清單、歷史數據、策略結果等

## 啟動方式

### 標準啟動
```bash
python -m streamlit run src/ui/web_ui.py --server.address=127.0.0.1 --server.port=8501
```

### 訪問地址
- 本地訪問: http://localhost:8501
- 網絡訪問: http://127.0.0.1:8501
